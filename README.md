# Document Question Answering System (RAG) 

## What it does

This project allows users to ask questions about a text document and receive answers generated by an AI model.

Instead of searching the entire document manually, the system finds the most relevant parts of the document and uses them as context for generating an answer.

The project uses Retrieval-Augmented Generation (RAG), which combines document retrieval with AI-powered answer generation.

## How it Works

### Step 1: Load the Document

The program loads a text file (`document.txt`) and splits it into smaller chunks.

This makes it easier to search through the document efficiently.

### Step 2: Retrieve Relevant Information

When a user asks a question:

* The question is converted into embeddings.
* The vector database searches for the most relevant chunks.
* The top matching chunks are selected as context.

### Step 3: Generate the Answer

The retrieved context is sent to the Groq AI model along with the user's question.

The AI generates an answer based only on the information found in the document.

## How to Run

### 1. Install Required Libraries

```bash
pip install langchain
pip install langchain-community
pip install langchain-text-splitters
pip install chromadb
pip install sentence-transformers
pip install groq
```

### 2. Create a Text File

Create a file named:

```text
document.txt
```

Add the content you want the AI to answer questions about.

### 3. Add Your Groq API Key

Replace:

```python
client = Groq(api_key="YOUR_GROQ_API_KEY")
```

with your actual Groq API key.

### 4. Run the Program

```bash
python app.py
```

### 5. Ask Questions

Example:

```text
Ask a question about the document:
```

Type your question and press Enter.
